01 — Overview

Abstract

Manual identification of resistors by reading color band codes is time-consuming and error-prone, particularly under variable lighting conditions or with aged components. This project presents an automated resistor classification system using convolutional neural networks (CNNs) to identify resistors by their color band patterns. We implemented and compared two architectures: a transfer-learned ResNet-18 model fine-tuned for resistor classification, and a lightweight custom CNN trained from scratch. Both models were trained with GPU acceleration using the Adam optimizer, learning rate scheduling, data augmentation, and regularization techniques. The classification pipeline operates in stages: a band count classifier first determines whether a resistor has 4 or 5 bands, followed by a dedicated value classifier for each type. Our dataset draws from proprietary lab-captured images and supplementary Kaggle sources, with all resistance values mapped against the IEC 60062 standard. Evaluation on held-out validation sets demonstrated near-perfect accuracy (98%+) for both architectures, confirming the effectiveness of the staged training pipeline. Real-world testing revealed a generalization gap under unseen lighting temperatures and non-standard backgrounds, motivating future work on expanded, more diverse datasets and ensemble prediction strategies.

02 — Method

System Architecture

The system uses a 3-step pipeline. A band count classifier routes each image to a 4-band or 5-band specialist model, which outputs the resistance value string.

03 — Evaluation

Key Findings

✅ Internal Success

Both models exceeded 98% accuracy on validation sets, confirming the effectiveness of the staged pipeline and augmentation strategy.

⚠️ Generalization Gap

Real-world testing revealed a performance drop on "unseen" images with different lighting temperatures or non-standard backgrounds — a target for future work.

🔭 Next Steps

Collect a more diverse dataset across manufacturers. Run both models in ensemble — averaging predictions and selecting the highest combined probability class.

📊 Custom CNN · Validation Accuracy

CNN Training Results — Prototype vs. Final

These donut charts compare validation accuracy across two iterations of the custom PyTorch CNN. The Prototype achieved 100% on band count classification, 98.38% on 4-band values, and 100% on 5-band values. After re-training the Final model on an expanded dataset, band count and 5-band accuracy held at 100% while 4-band accuracy settled at 97.27% — indicating the larger, more diverse dataset introduced harder edge cases that slightly increased confusion among similar-looking 4-band resistor values.

📈 ResNet-18 · Dataset Scaling

ResNet-18 Training Results — Prototype → V1 → Final

This chart tracks how the ResNet-18 transfer learning model improved as the dataset scaled from roughly 1,200 images (Prototype) to 4,000 images (V1) and finally 16,000 images (Final). The band count classifier was perfect at every stage. The 4-band value classifier improved from 97% → 98.51% → 99.03%, and the harder 5-band task climbed from 87% → 95.67% → 97.16% — clearly demonstrating that dataset size was the primary bottleneck for the more difficult 5-band classification.

📉 Per-Class · 101 Classes · 15,288 Images

Per-Class Accuracy Bar Chart

This chart shows classification accuracy broken down by all 101 resistance value classes, comparing 4-band (blue) and 5-band (white) resistors. The red dashed line marks the 92.35% overall accuracy across the full 15,288-image test set. The majority of classes exceed 95%, and many hit 100%. The handful of lower-performing classes — some dipping below 50% — correspond to resistance values whose color band sequences are visually similar (e.g., values sharing brown, black, and red bands), and are the primary target for future ensemble and augmentation improvements.

🔲 Confusion Matrix · Overall System

Full System Confusion Matrix (101 Classes)

The full 101-class confusion matrix for the complete classification pipeline on 15,288 test images. Green squares along the diagonal represent correct predictions; red squares indicate misclassifications. The strong diagonal with only sparse off-diagonal red entries confirms the system's 92.35% overall accuracy. The denser cluster of misclassifications in the lower-right region corresponds to 5-band resistors with closely spaced resistance values — a known challenge in color-based classification at this scale.

🔵 Confusion Matrix · 4-Band Classifier

4-Band Value Classifier Confusion Matrix

The 4-band value classifier achieved 99.95% accuracy on 2,010 test images across all 4-band resistor classes. The near-perfect diagonal with virtually zero off-diagonal entries demonstrates that the model has effectively learned the color band encoding for 4-band resistors. The isolated single misclassification visible is consistent with the inherent visual ambiguity between certain color bands — particularly under variable lighting — rather than any systematic model failure.

🔵 Confusion Matrix · 5-Band Classifier

5-Band Value Classifier Confusion Matrix

The 5-band value classifier achieved 98.63% accuracy on 3,440 test images (3,393 correct). The confusion matrix shows a clean diagonal across all 5-band resistance classes, with only a small number of off-diagonal misclassifications. The slight spread observed in a few classes — notably between 1 kΩ, 2 kΩ, and similar values — reflects the visual similarity between bands in the 5-band encoding scheme. This is lower than the 4-band model, consistent with the expectation that more color bands introduce more opportunity for per-band ambiguity.
